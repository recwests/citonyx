## Summary
Erlin data across 500+ brands: brands with only one source type achieve 18% average AI coverage. Two sources = 35%, three = 58%, five or more = 78%. Source diversity compounding is a core GEO multiplier. Digital PR, review platforms, YouTube presence, and Reddit participation are now core tactics.

## Details
The compounding effect means brands cannot rely on a single citation channel. The most effective brands build presence across: owned content (blogs, research), review platforms (G2, Capterra, Trustpilot), community platforms (Reddit, Quora), YouTube (transcripts and captions), and digital PR (news mentions). Each additional source type adds non-linear coverage gains. For the Citonyx testbed: prioritize building 3-5 distinct source types before deepening any single channel. The jump from 1 to 3 sources (+40pp) is the highest-leverage move, larger than the jump from 3 to 5 (+20pp).
