## Summary
Claude Skills (launched May 2026) package SEO workflows into reusable .md instruction files. Combined with MCP connectors to Ahrefs, GSC, GA4, and Slack, Claude can run technical audits, keyword clustering, content gap analysis, schema generation, and E-E-A-T scoring — cutting audit-to-action from days to under 30 minutes.

## Details
200K+ token context window allows uploading an entire Screaming Frog crawl or year of Ahrefs exports. MCP (Model Context Protocol) connects Claude to live data from Ahrefs, GSC, GA4 without manual CSV exports. Claude Code runs in-terminal with file access and MCP servers. Claude Skills are saved SOPs invoked by trigger phrase. Top open-source libraries: claudeseoskill.com (Alex Labat), AgricIDaniel (25 sub-skills + 18 agents), Hawk Academy (37 free skills). Replaces ~$400/month of paid SaaS tooling.
