## Summary
Mid-market HR SaaS (DR 78) lost 22% YoY lead velocity as AI summarized their content without citation. FuelOnline GEO Nexus framework: (1) Wikidata entity mapping linking software features to global entities (ERP, payroll), (2) original performance metrics from product engineers ('API reduces latency by 41%'), (3) nested FAQPage inside SoftwareApplication schema with SameAs to G2/Capterra. Results: +315% citations (12→50), 28 top-3 mentions for comparison queries, +140% demo requests.

## Details
4-month deployment. MQLs increased 55→132/month. Time on site improved 1m12s→3m45s (+212%). Original performance metrics made the client the sole primary source for specific data points — AI engines had to cite them. Nested schema with SameAs attributes linking to third-party review entities created semantic trust loop. Before: high-traffic 'Ultimate Guides' were scraped without citation. After: content re-engineered so AI engines had 'no choice but to cite' the data.
