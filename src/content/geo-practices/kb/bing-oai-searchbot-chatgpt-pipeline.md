## Summary
ChatGPT Search draws live results from Bing's index, not Google's. If a page isn't in Bing, ChatGPT cannot cite it regardless of Google ranking. Four-gate funnel: Bing index → OAI-SearchBot access → retrieval → citation selection; only ~15% of eligible pages get quoted.

## Details
Seer Interactive found 87% of ChatGPT Search citations matched Bing's top organic results (vs 56% for Google) across 500+ citations. Bing Webmaster Tools offers a free AI Performance dashboard (public preview Feb 2026) showing actual citation events, cited URLs, and the grounding sub-queries used — the only first-party AI-citation analytics tool. IndexNow (Bing/Yandex/Naver) pushes new/updated URLs into Bing within minutes vs weeks, so fresh content can reach ChatGPT in 1–3 weeks. Import from Google Search Console for one-click verification.
