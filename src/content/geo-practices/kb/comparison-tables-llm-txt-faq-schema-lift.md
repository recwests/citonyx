## Summary
Erlin's controlled testing across 500+ brands tracked across ChatGPT, Perplexity, Gemini, and Claude found: comparison tables drive +34% coverage lift (14 day median), llm.txt files +32% (14 days), FAQ schema +28% (21 days). Each missing structured data element represents a 6-8% coverage gap. Brands with all five elements reach 60-80% prompt coverage.

## Details
Time to impact is consistent: comparison tables 12-16 days, llm.txt 11-17 days, FAQ schema 18-24 days. These are controlled-tested results, not correlations. JS-rendered content and missing schema.org markup contribute to the coverage gap. Cloudflare recently changed default config to block AI bots — brands using Cloudflare may have lost AI crawler access automatically. The Erlin data provides per-element granularity missing from general schema studies: not all structured data is equal — comparison tables and llm.txt have the fastest time-to-impact and highest absolute lift.
