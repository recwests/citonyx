# Citonyx

A GEO (Generative Engine Optimization) testbed that measures whether AI answer engines actually cite it.

Built with [Astro](https://astro.build/) + [Tailwind CSS](https://tailwindcss.com/), deployed on Cloudflare Pages.

## Commands

All commands are run from the root of the project:

| Command           | Action                                     |
| :---------------- | :----------------------------------------- |
| `npm install`     | Install dependencies                       |
| `npm run dev`     | Start local dev server at `localhost:4321` |
| `npm run build`   | Build the production site to `./dist/`     |
| `npm run preview` | Preview the build locally before deploying |
| `npm run check`   | Check the project for errors               |
| `npm run fix`     | Run ESLint and format code with Prettier   |
