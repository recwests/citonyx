## Summary
Google retired FAQ rich results on May 7, 2026, but FAQPage schema is still parsed and consumed by AI engines for citation extraction. OpenHermit reports 28-40% citation lift; Ranqo reports FAQPage schema increases Perplexity direct-answer retrieval by +31%. The visible rich result is gone, but AI extraction value remains.

## Details
FAQPage schema remains valid Schema.org — Google still parses it for page understanding. The May 7 retirement only removed the visible rich result from SERPs, not the schema's value for AI extraction. Frase reported 3.2x AI Overview inclusion for FAQ pages. Goodie's 2.2-million prompt analysis found FAQ schema increased Perplexity direct-answer retrieval by +31%. Google explicitly warns against invisible FAQ tricks — schema must match visible content.
