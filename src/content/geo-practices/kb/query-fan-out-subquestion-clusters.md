## Summary
Gemini 3 (Jan 2026) uses query fan-out, increasing cited sources 32% and replacing 42% of previously-cited domains. A page ranking #5 for the main keyword but best-answering a sub-question can now win. Depth on one URL beats thin spread across many.

## Details
Decompose seed query into 5–10 sub-questions (from People Also Ask, Reddit, AI Overview follow-ups), make each an H2 with a 40–80 word answer. Bing's AI Performance report exposes the actual grounding sub-queries for your domain — the only place to see fan-out retrospectively. Conductor 2026: pages ranking for fan-out sub-queries are 161% more likely cited in AIO than head-term-only pages.
