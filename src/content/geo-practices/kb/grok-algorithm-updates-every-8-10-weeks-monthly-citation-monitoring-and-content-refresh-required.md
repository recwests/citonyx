## Summary
Goodie's algorithm analysis shows Grok receives significant updates approximately every 8-10 weeks. Citation patterns can shift materially with each update. Monthly citation monitoring and quarterly content audits are the minimum cadence. Grok's 90% of winning citations answer the query within the first 100 words (BLUF principle).

## Details
Key Grok-optimization signals: answer-first formatting (first 100 words), conversational query matching (full-sentence H2 headers like users ask on X), freshness via dateModified, X (Twitter) co-citation, and verified X account status as a trust signal. Grok's defining advantage is real-time access to X's live content stream. Entity SEO is foundational — Grok's knowledge graph integration means brands with clearly defined entities (Wikipedia, Wikidata, sameAs schema) are more likely cited.
