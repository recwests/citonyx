## Summary
Erlin's controlled testing across 500+ brands tracked across ChatGPT, Perplexity, Gemini, and Claude found: comparison tables drive +34% coverage lift (14 day median), llm.txt files +32% (14 days), FAQ schema +28% (21 days). Each missing structured data element represents a 6-8% coverage gap. Brands with all five elements reach 60-80% prompt coverage.

## Details
Time to impact is consistent: comparison tables 12-16 days, llm.txt 11-17 days, FAQ schema 18-24 days. These are controlled-tested results, not correlations. AI parsing success rates: static HTML with schema = 94%, plain HTML no schema = 68%, JavaScript-rendered = 23%, PDF = 7%. Each missing structured data element represents a 6-8% coverage gap. Brands with all five structured attributes reach 60-80% prompt coverage vs 23-35% for those missing all. JS-rendered content fails AI parsing 77% of the time — if critical pages load content dynamically, AI crawlers read empty templates. Content with tables gets cited 2.5x more often than plain text. The Erlin data provides per-element granularity missing from general schema studies.
